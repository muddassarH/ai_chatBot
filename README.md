# AI CHAT BOT 
##  TO RUN FRONTEND 
# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) Node.js v20.12.0 (LTS)
 
 
# Getting started
- Clone the repository
```
git clone  <git url> <project_name>
```
- Install dependencies
```
cd <frontend>
npm install
```
- run the project in development mode
```
npm run dev
```
  Navigate to `http://localhost:3000`
